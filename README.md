# Controle de Ativos - Oba Hortifruti

Sistema para consulta e gestão de ativos patrimoniais e equipamentos, integrado com Supabase e suporte a planilhas Excel.

## 🚀 Funcionalidades

- **Consulta Rápida:** Filtros por placa, número da loja e descrição.
- **Detalhes do Ativo:** Visualização completa de valores, depreciação e localização ao clicar na linha.
- **Área Administrativa:** Protegida por senha para gestão de dados.
- **Importação de Dados:** Carga em lote via arquivos Excel (.xlsx).
- **Exportação:** Geração de relatórios em Excel dos ativos cadastrados.

## 🛠️ Tecnologias Utilizadas

- [React](https://reactjs.org/) + [Vite](https://vitejs.dev/)
- [TypeScript](https://www.typescriptlang.org/)
- [Tailwind CSS](https://tailwindcss.com/)
- [Supabase](https://supabase.com/)
